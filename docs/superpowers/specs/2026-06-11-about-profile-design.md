# 关于我技术成长档案设计

## 目标

将“关于我”板块中重复技能域信息的标签网格，替换为理性、专业的技术成长档案。内容不展示兴趣爱好，不使用能力百分比，重点表达当前定位、工作方式和未来技术路线。

## 内容结构

- 当前定位：AI 应用开发、全栈原型、数据技术基础。
- 工作方式：问题拆解、快速验证、工程化迭代。
- 未来方向：AI Agent / Multi-Agent、Spark 分布式计算、深化 Hadoop 生态、大数据工程与实时处理。
- 状态行：`FOCUS 2026+ / AGENT SYSTEMS / DISTRIBUTED DATA`。

## 视觉方向

- 纯黑、白、灰配色，不使用绿色或其他强调色。
- 使用细分隔线、编号、字重和留白建立层级。
- 三个区域属于同一个连续档案版面，不做圆角标签、进度条或独立卡片。
- “未来方向”占据更大视觉权重，并以纵向路线表达持续深入关系。
- 桌面端保持左右双栏；窄屏改为单栏，路线文字不溢出。

## 范围

仅修改 `index.html` 中关于我板块的标签结构，以及 `src/styles.css` 中对应样式。保留独立“技能域”章节和其他页面内容。
