# 个人主页 2.0 — GPT 生图 Prompt

```
请生成一张个人主页首屏设计稿，作为网页设计参考（非成品截图）。

【整体风格】
- 瑞士平面设计风（Swiss Design）：严格网格系统、大字号标题、高对比度、大量留白，所有文字板块左对齐，拒绝居中排版
- 黑白为主色调（背景白 #FAFAFA，文字黑 #1A1A1A），金色 #C9A040 仅用于关键强调处（下划线、小标记、星标），整体颜色占比不超过 5%
- 磨砂玻璃质感：顶部导航栏 backdrop blur 半透明，内容区卡片轻微玻璃感
- 专业、干净、有品味，像一个懂技术的设计师的作品集，不花哨不炫技

【布局结构 — 单页长卷，从上到下，所有内容左对齐】

1. 顶部导航栏：磨砂玻璃底（白色 60% 透明度 + backdrop blur），固定在页面顶部。左侧"Vincent"粗体黑字，右侧四个链接（首页 项目 博客 联系），当前页"首页"下方有一道金色短下划线。导航栏底部 1px 浅灰分割线。

2. Hero 首屏：严格左对齐。大标题"翟伟鑫"（字号约 72px，极粗体，黑色）紧贴左侧网格线；下一行"Vincent"（字号约 48px，常规字重，深灰）；再下一行"大数据管理与应用 · AI Agent 与 3D 可视化"（字号约 18px，灰色）。整段无任何居中元素，右侧大面积留白。标题下方一道金色短横线（宽度约标题的 1/3），CTA 文字"查看项目 ↓"（黑色）。Hero 高度占视口约 90%，文字块沉在偏下 1/3 处，上方大量留白。

3. 关于我：左对齐。小标签"关于"（全大写，字间距拉开，金色），下方 3-4 句中文简介段落（行宽约 600px，深灰色文字，行高 1.8）。

4. 项目展示：左对齐。小标签"项目"（全大写，字间距拉开，金色）。下方 2 行 × 3 列的卡片网格（共 6 张）。每张卡片：磨砂玻璃底（白色 50% 透明度 + 轻微阴影 + 极细边框），左上角金色星标（★，仅重点项目），项目名粗体黑色，一行简述灰色小字，底部技术标签（等宽字体，浅灰底黑字小标签）。卡片悬停时背景变实（白色 80% 透明度）、微微放大。

5. 技术博客：左对齐。小标签"博客"（全大写，字间距拉开，金色），左侧一条金色竖线装饰，右侧文章列表 3-5 条。每条：标题（黑色粗体）、日期（灰色小字）、一行摘要（灰色）。条目之间细线分隔。

6. 联系方式 & 页脚：左对齐。Email 地址、GitHub 链接（黑色文字）。下方一句金句"AI 不是替代人，而是放大人的创造力。"（金色，斜体）。最底部版权行灰色小字。

【排版细节】
- 最大内容宽度 1024px，内容区整体左对齐但页面居中
- 标题用 Inter 或类似无衬线字体，正文用 Noto Sans SC
- 大标题字间距略收紧（tracking-tight），小标签字间距拉开（tracking-widest）
- 板块之间留白充足（约 120-160px 垂直间距）
- 字号层级：72px → 48px → 24px → 18px → 14px → 12px，阶梯分明

【交付格式】
一张宽约 1440px、高约 4000px 的纵向长卷设计稿，包含上述所有板块。
```
